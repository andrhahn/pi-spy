# pi-spy api server

pi-spy backend vps api server

##### Setup
    Install Node
    Install Mongo
    
    Then run:
    npm install

#### Run
    Set:
    NODE_ENV: development
    PORT: 1337
    CORS_ORIGIN: localhost
    MONGODB_URI: mongodb://localhost/replogger
    
    Then run:
    npm run start or node dist/app.js
