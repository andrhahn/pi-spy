# pi-spy

Camera functions for Raspberry Pi

### Getting Started

##### Setup
    sudo apt-get update
    sudo apt-get upgrade
    sudo rpi-update
    sudo reboot

    sudo apt-get install python-picamera
    
    sudo apt-get install netatalk
    sudo apt-get install avahi-daemon
    
    npm install grunt-cli -g
    npm install
